EasyML

Welcome to EasyML!

EasyML helps connect you with powerful machine learning tools for a variety of applications. As machine learning becomes more complex, it becomes more difficult to use these tools. However, EasyML helps to streamline this process by providing a user-friendly interface for these ML tools.

Protein Classification

Proteins are crucial for biological processes, with a wide variety of proteins performing different functions. By typing an amino acid sequence, you can predict what type of protein that it is!
Try the program here.

Pet Classification

Not sure if your pet is a dog or a cat?
Use this tool to predict which animal a pet is from a single image! This machine learning tool uses a convolutional neural network to classify pictures to dogs or cats.
